# alura-store-analisis
#  Análisis de Datos - Alura Store

Proyecto de análisis de datos realizado como parte del programa **Oracle Next Education - Alura Latam**.

##  Objetivo del análisis

Analizar las ventas de la tienda Alura Store para identificar patrones, tendencias y oportunidades de mejora en el negocio.

##  Estructura del proyecto
alura-store-analisis/
│
├── Colab_Mau.ipynb
└── README.md

##  Análisis realizados

Durante el proyecto se realizaron los siguientes análisis:

- Limpieza de datos
- Exploración de datos
- Visualización de ventas
- Identificación de productos más vendidos
- Análisis de ingresos por categoría

##  Ejemplos de gráficos

El notebook incluye gráficos como:

- Ventas por categoría
- Productos más vendidos
- Distribución de ingresos

##  Cómo ejecutar el proyecto

1. Descarga el archivo `.ipynb`
2. Ábrelo en **Google Colab**
3. Ejecuta las celdas para reproducir el análisis

##  Tecnologías utilizadas

- Python
